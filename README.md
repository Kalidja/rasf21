# Тестовое задание

Этот проект представляет собой тестовое задание на Django.  
Ниже приведена нструкция по установке, настройке и запуску проекта локально через `runserver`.

## 1. Подготовка среды выполнения

Установить:

- Python 3.10+
- pip
- Git

## 2. Клонирование репозитория

git clone https://github.com/Kalidja/rasf21.git  
cd rasf21



## 3. Виртуальное окружение

python3 -m venv .venv  
source .venv/bin/activate
